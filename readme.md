# Linux

## Commands

`ls` - List Directory
`chmod` - Change Modification
`useradd` - Add User
`groupadd` - Add Group
`usermod` - Modify User
`su` - Switch User
`passwd` - Set Password
`touch` - Update last time file was updated
`pwd` - Print Working Directory
`cd` - Change Directory
`cat` - Show Contents of File
`mkdir` - Make Directory
`chown` - Change Ownership
`env` - Show Variables
`rm` - Remove
`mv` - Move
`cp` - Copy
